# Mohammed Zidan C — Portfolio

Personal portfolio website for Mohammed Zidan C, an aspiring VLSI/RTL Design engineer and B.Tech student at SRM Institute of Science and Technology, Chennai.

🌐 **Live site:** [mohammedzidanc.vercel.app](https://mohammedzidanc.vercel.app)

---

## About

Built to showcase projects, education, skills, and certifications in an immersive, animation-heavy dark-theme design. Every section is animated. Every interaction has feedback.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 14 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS |
| Animation | GSAP + Framer Motion |
| Smooth Scroll | Lenis |
| Fonts | Clash Display, Syne, General Sans, JetBrains Mono |
| Deployment | Vercel |

---

## Sections

- **Hero** — Animated name reveal, aurora blobs, constellation canvas, scramble text
- **About** — Bio, animated stat counters, infinite marquee
- **Skills** — Categorized tag cloud with staggered scroll reveal
- **Education** — Animated vertical timeline (4 entries)
- **Projects** — Staggered card reveal with holographic hover effects (4 projects)
- **Certifications** — Grid of all 10 certifications with PDF links
- **Contact** — Email, GitHub, LinkedIn

## Running Locally

```bash
# Clone the repository
git clone https://github.com/MohammedZidanC/my-portfolio.git
cd my-portfolio

# Install dependencies
npm install
# or
pnpm install

# Start development server
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## Building for Production

```bash
npm run build
npm run start
```
