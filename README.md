## ✨ Overview

This is the personal portfolio of **Mohammed Zidan C** — an aspiring VLSI/RTL Design engineer and B.Tech student at SRM Institute of Science and Technology, Chennai.

The site is designed with a maximalist, immersive dark-theme aesthetic. Every section features scroll-driven animations, staggered reveals, holographic hover effects, and micro-interactions that make the experience feel alive and premium.

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| **Framework** | Next.js 14 (App Router) |
| **Language** | TypeScript |
| **Styling** | Tailwind CSS |
| **Animation** | GSAP + Framer Motion |
| **Smooth Scroll** | Lenis |
| **Fonts** | Clash Display, Syne, General Sans, JetBrains Mono |
| **Deployment** | Vercel |

## 💻 Running Locally

```bash
# Clone the repository
git clone https://github.com/MohammedZidanC/Portfolio.git
cd Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## 📦 Building for Production

```bash
npm run build
npm run start
```
