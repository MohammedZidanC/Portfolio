## ✨ Overview

This is the personal portfolio of **Mohammed Zidan C** — an aspiring VLSI/RTL Design engineer and B.Tech student at SRM Institute of Science and Technology, Chennai.

The site is designed with a maximalist, immersive dark-theme aesthetic. Every section features scroll-driven animations, staggered reveals, holographic hover effects, and micro-interactions that make the experience feel alive and premium. It's not just a portfolio — it's an experience.

---

## 🎨 Design Philosophy

The design draws inspiration from premium, editorial-style websites with a focus on:

- **Dark Mode First** — A carefully crafted dark theme with layered backgrounds (`--bg-primary`, `--bg-secondary`, `--bg-tertiary`) that create depth and visual hierarchy without eye strain.
- **Cinematic Animations** — Every element enters the viewport with purpose. From word-by-word text reveals to staggered card animations, each transition is carefully timed to feel natural and cinematic.
- **Holographic Effects** — Project cards feature a holographic shimmer on hover, giving the interface a futuristic, premium feel.
- **Aurora & Constellation Canvas** — The hero section features animated aurora blobs and a live constellation particle canvas that react to user interaction.
- **Typography System** — Four carefully selected typefaces work in harmony:
  - **Clash Display** for bold headings
  - **Syne** for display type
  - **General Sans** for body copy
  - **JetBrains Mono** for code and technical labels
- **Micro-Interactions** — Hover effects, magnetic buttons, smooth transitions, and subtle feedback on every interactive element.

## 🎯 Performance & Optimization

- **Server-Side Rendering** — Next.js App Router ensures fast initial page loads with server-rendered HTML
- **Lazy Loading** — All project images use `loading="lazy"` to defer off-screen content
- **Intersection Observer** — Animations are triggered only when elements enter the viewport, keeping the initial render lightweight
- **Optimized Fonts** — Custom fonts are loaded via `next/font` for zero-flash rendering
- **Image Optimization** — All images served through Next.js `<Image>` component with automatic format selection and responsive sizing
